# Gestion de Garanties des Soumissions (Bid Guarantees Management)

> A secure web application for tracking and managing bid guarantees, calls for tenders, and associated financial documents.

## Overview
This project is a comprehensive PHP/MySQL application developed to digitize and manage the lifecycle of bank guarantees related to business tenders. It centralizes data for agencies, banks, bidders, and currencies, providing a clear dashboard and secure document management system for financial compliance. 

## Technologies Used
* **Backend:** PHP
* **Database:** MySQL (Relational Database Management)
* **Frontend:** HTML, CSS, JavaScript 
* **Asset Management:** File uploading and handling for PDF documents (Amendments, Authentications, Guarantees, Liberations)

## Key Features
* **Secure Authentication:** Role-based login and session management for users.
* **Tender Management (Appels d'Offres):** Create, track, and update details for various calls for tenders.
* **Guarantee Tracking:** Full lifecycle management of bid guarantees, including logging issuing banks, currencies, and release statuses (libérations).
* **Document Archiving:** Automated secure uploading and linking of PDF documents (authentications, amendments) to their respective tender records.
* **Entity Management:** Admin interfaces to manage structures, agencies, countries, and participating bidders.
